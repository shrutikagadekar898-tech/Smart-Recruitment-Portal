# Smart Recruitment Portal

## Project Overview

Smart Recruitment Portal is a Java-based recruitment management system that connects Candidates and Companies through a centralized platform. The system provides role-based access for Candidates, Companies, and Admins to manage job applications, interviews, and job offers efficiently.

## Features

### Candidate Module
- Candidate Registration & Login
- Update Profile
- Upload & View Resume (PDF)
- Search Jobs by Skills, Experience, Salary, and Location
- Apply for Jobs
- View Applied Jobs
- Accept or Reject Job Offers

### Company Module
- Company Registration & Login
- Post New Jobs
- Update & Delete Posted Jobs
- Search Jobs using Filters
- Schedule Interviews (Online / Offline / HR / Technical)
- Generate Job Offers
- Manage Interviews and Offers

### Admin Module
- Admin Login
- Dashboard Statistics
- View Total Users, Companies, Jobs, and Offers
- Manage Users (Add, Update, Delete)
- Manage Companies (Add, Update, Delete)
- View Jobs and Applications

### Email Notifications
- Job Application Confirmation
- Interview Schedule Notification
- Job Offer Notification
- Offer Status Updates
